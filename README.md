# Astro ESI Demo

## Run Demo

Build an run using:

```sh
sudo ./rebuild-server-and-run.sh
```

Open the follwoing URLs:

| URL                          | Description                    |
| ---------------------------- | ------------------------------ |
| http://localhost:8080/?id=33 | Shell Server with Fragment ESI |
| http://localhost:8081/?id=33 | Fragment Server (Widget)       |
